# 1. Product Overview

## Description

Cogna is an API-first AI subscription marketplace that allows customers and businesses to purchase, manage, and resell premium AI subscriptions such as ChatGPT Plus, Gemini Pro, Claude Pro, Cursor Pro, Perplexity Pro, GitHub Copilot, and other AI services.

The platform integrates with reseller providers and payment gateways to automate fulfillment while exposing a REST API for developers.

---

# 2. Problem Statement

Purchasing AI subscriptions is often difficult due to payment restrictions, regional availability, and the lack of centralized management.

Businesses that want to resell AI subscriptions also need a secure API instead of integrating directly with multiple providers.

Cogna solves these challenges by providing one secure platform for customers and developers.

---

# 3. Goals

- Simplify AI subscription purchases.
- Automate order fulfillment.
- Provide developer-friendly APIs.
- Support multiple providers.
- Deliver subscriptions quickly and securely.

# 8. MVP Scope

Included

- Authentication
- Products
- Orders

## Payments

- Multi-gateway: Paystack + Monnify (extensible)
- Per-product payment gateway assignment
- Payment verification per gateway
- Payment history
- Refund management (Admin)

## Per-Product Provider API

- Each product is linked to a specific provider and its API config
- Supports multiple providers simultaneously
- Provider API key and base URL stored per-provider

- Admin Dashboard
- Developer API
- Akunding Integration

Excluded

- Mobile Apps
- Affiliate System
- Wallet
- Referral Program
- Multiple Providers
